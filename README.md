Image-Processing
================

Using back camera of mobile,app can "train" and "search" the image of any person with the output of his/her name if image exists in database and if not,then take image with "train" option and then search.You can use this sorce code for other purposes like security system for mobile like locks or tagging persons in pictures.
This app  uses Open-CV-library-4.4.2 files.Just open eclipse and import project and Run it.
