Image-Processing
================

Using back camera of mobile,app can "train" and "search" the image of any person with the output of his/her name if image exists in database and if not,then take image with "tarin" option and then search.You can use this sorce code for other purposes like securty system for mobile like locks or tagging persons in pictures.
The app is using Open-CV-library-4.4.2 files.Just open eclipse and import project and Run it.
